DESCRIPTION:

This a simple blog application writen in Spaghetti PHP and the following steps are needed to run it effectively
on your device

Step 1: Clone the files to your device and create a folder for it.
Step 2: Make sure you move the folder to your htdocs.
Step 3: Ensure that you have Xampp install to run a localhost.
Step 4: Create a Mysql database and import the db file attached to this project.
Step 5: Go to your browser and type localhost, it should show you the folder of this project.
Step 6: Click on the folder and it automatically shows up on your browser.
Step 7: Feel free to register and log in : )
